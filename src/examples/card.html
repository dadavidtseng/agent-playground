Integration Notes

  • Inline SVG: Insert inline SVG markup inside the .char-card__avatar element to ensure it inherits size and can be styled via CSS. Example shown for card 2.
  • Image: Use <img> inside .char-card__avatar for raster portraits. Ensure alt text is provided for accessibility. Set aria-hidden="true" on avatar wrappers if alt text duplicates visible text.
  • Data attributes: Stats expose a simple schema: data-stat-type (string) and data-stat-value (number). Example: data-stat-type="hp" data-stat-value="84". JavaScript can query these for dynamic updates.
  • ARIA: The card uses semantic elements (article, header, footer). Use role="list" on the container if cards are purely presentational items within a list.
