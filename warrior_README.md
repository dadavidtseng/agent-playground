Export recommendations:
- Export as PNG-24 at 64x64 and 128x128.
- Use nearest-neighbor / disable resampling to preserve pixel edges. In browsers or editors, enable 'image-rendering: pixelated' or set interpolation to 'Nearest Neighbor'.
- Ensure PNG export is 24-bit (no palette reduction) to avoid color banding.

Optimize:
- The SVG is already integer-aligned and minimal; you may run SVGO to remove optional metadata and further minimize file size.

Purpose:
- This SVG is a single-source pixel-art warrior placeholder composed of 1x1 rect "pixels" in a 64x64 viewBox. Groups: body, armor, weapon, accessories are editable.
