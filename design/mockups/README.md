Mockups

This folder contains references for mockup files demonstrating breakpoints and component states. Per task restrictions we do not include finalized SVG assets, but provide a placeholder SVG that designers can replace in Figma or export from design tools.

Planned files:
- character-card-breakpoints-and-states.svg (placeholder describing layout)

Notes for designers:
- The SVG should include three artboards: mobile (375x800), tablet (800x1024), desktop (1400x900).
- Each artboard should render 3 example cards showing: default, hover, active, disabled states.
- Use token-based colors and spacing when building the SVG so it can be imported into Figma with consistent styles.
